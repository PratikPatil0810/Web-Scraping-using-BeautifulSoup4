# Hacker News Scraper 

This project scrapes Hacker News using **Python + BeautifulSoup** and saves the data to a CSV file.

## Features
- Scrapes title, rank, points, author, comments
- Uses requests + BeautifulSoup
- Saves output as CSV

## Tech Stack
- Python
- BeautifulSoup
- Requests
- Pandas

## How to Run

pip install -r requirements.txt
python hacker_news.ipynb
